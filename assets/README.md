# assets/

Static image assets for the portfolio (logos, illustrations, exported charts).

## What goes here

- Brand or profile imagery (optional).
- Exported benchmark charts once the clean release run is complete.
- Any diagram images you prefer over the CSS-built architecture diagram.

## Guidelines

- Use web-friendly formats: **PNG**, **JPG/JPEG**, **WebP**, or **SVG**.
- Keep file sizes reasonable (compress large PNGs/JPGs).
- Reference assets with **relative paths** so the site stays portable and offline-capable, e.g.:
  ```html
  <img src="assets/your-image.png" alt="Descriptive text" />
  ```
- Always provide meaningful `alt` text for accessibility.
