# Pipeline Metric Comparison Dashboard v1

Open `dashboard.html` in a browser. The dashboard is static and local: no server, CDN, API call, or build step is required.

It summarizes the metric-comparison findings from `metric_comparison_findings_report_v1`, including model pass/fail rates, BERTScore v2, ROUGE-L, chrF, ROSCOE reasoning alignment, tolerance layers, numeric relative-error findings, and large numeric failure gates.

For LinkedIn, publish the folder through GitHub Pages or another static hosting option, then link to the dashboard.
